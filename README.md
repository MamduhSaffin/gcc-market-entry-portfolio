## Deploy

### GitHub Pages
1. Go to repository Settings → Pages
2. Select "Deploy from a branch"
3. Choose "main" branch
4. Your site will be live at: https://MamduhSaffin.github.io/gcc-market-entry-portfolio/

### Vercel (Recommended)
1. Go to https://vercel.com
2. Import this GitHub repository
3. One-click deployment

## Contact Form Setup

To enable the contact form, choose one:

1. **Formspree** (Recommended)
   - Go to https://formspree.io
   - Create account and form
   - Update form endpoint in contact.html

2. **Netlify Forms**
   - Deploy on Netlify
   - Forms work automatically

3. **Microsoft Forms**
   - Create form at forms.microsoft.com
   - Embed in contact page
